# Slovak case trainer / Тренажёр склонения существительных в Словацком языке
A simple trainer for memorizing patterns (vzor) of declension of nouns by case and number in the Slovak language.
<br>
Предельно простой тренажёр на запоминание склонения образцов (vzor) склонения имён существительных по падежам и числам в Словацком языке.

# Requirements / Требования
- python3
- tkinter

# Installation / Установка
- ```git clone https://github.com/Nikita-14365/slovak_case_trainer.git```
- ```pip install tkinter```

# Usage / Использование
- ```cd slovak_case_trainer```
- ```python3 main.py```
<br>
or other name of your utility for calling python interpreter.
<br>
или другое название утилиты для вызова интерпретатора python.
